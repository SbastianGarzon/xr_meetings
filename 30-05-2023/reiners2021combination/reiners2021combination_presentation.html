Interpretation of XR generated data (33%)

  • What can we say about users based on XR generated data ?
  • “Combining AI and XR provides an opportunity to better interpret XR dynamics by developing and objective approach for assessing user skill and performance” Reiners et al. (2021)

Skill evaluation

  • Providing metrics and performance feedback in a surgical simulator - Sewell et al. (2008)

  • XR: Virtual surgery (drilling technique - Bone removal)

Proposed questions

  • “Training AI” papers consider “AI Agents” (e.g., Autonomous cars) as “users” experiencing virtual environments. Should we consider these examples as an “AI-XR combination” ?

  • Do you know any project that could be considered an “AI-XR combination”? (Point cloud labeling)

  • Can machine algorithms learn useful behavior from observing human user interactions with the system? Can they learn semantic relationships (e.g., spatial grouping)? Can this be enhanced by enabling more immersive interactions (e.g., positional and/or eye tracking)? It would seem that the more user interactions that can be captured, the more that could be learned from them. Skarbez et al. (2019)
